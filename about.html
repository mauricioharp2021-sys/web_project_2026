Project Overview

This site is being built step-by-step using semantic HTML5 structure as part of ICT 420.

The focus of this project is on structure and meaning rather than visual design. By using semantic elements, the site is easier for both users and machines to understand.

As the course continues, this site will expand to include additional pages, richer content, and more advanced semantic features.
