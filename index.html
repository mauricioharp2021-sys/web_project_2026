Project Overview

Semantic Web Development

What this site is

Structure first

This site is being built step-by-step using semantic HTML5 structure.

This website is my step-by-step project for ICT 420. I’m building it using semantic HTML so the structure makes sense before I worry about design.

Right now the goal is to practice writing clean HTML: using headers, sections, and navigation correctly so someone can skim the page and still understand it.

I’m also thinking about accessibility as I build. If the structure is clear, it helps both people and tools like screen readers understand what’s on the page.

Current goal

Consistency across pages

My current goal is to keep the site organized and consistent across pages. Each page should have the same navigation and a clear layout that doesn’t rely on styling.

I’m focusing on headings and sections first so the outline is easy to follow. That way, when I add more content later, I won’t have to rebuild the structure.

What I will add later

Content and metadata

Later in the semester I plan to expand the content with course topics like metadata, linked data, and schema concepts. I also want to add more examples that show what I’m learning each week.

I will also improve the metadata in the head section as we cover more topics and add more pages. I’m keeping descriptions and keywords relevant so the site stays clear for both users and machines.

When we start CSS, I’ll work on design and layout. For now, I’m keeping it HTML-only so the meaning and structure come first.

Tools and Workflow

VS Code, Git, and GitHub

My setup

How I’m building the site

I’m building this site using VS Code and GitHub. Writing locally and pushing updates helps me track changes and see my progress through commit history.

GitHub Pages makes it easy to publish the site as a real website. It’s helpful because I can test navigation and see how the pages work in a real browser.

My current setup is macOS. I’m keeping the workflow simple: edit, save, commit, push, and check the live site.
