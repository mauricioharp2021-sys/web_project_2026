Project Overview

Semantic Web Development

What this site is

This site is being built step-by-step using semantic HTML5 structure. The goal is to organize the page in a way that makes sense for both users and machines.

Note: This project is focused on structure and meaning first—design comes later.

Current goal

Right now I’m focusing on clean structure, working navigation, and accessibility basics before adding styling later in the course.

What I will add later

Content and metadata

Later I plan to add more content to each page, expand sections, and keep improving the structure as we learn more HTML. Once CSS is introduced, I’ll start working on design and layout.

I’m using metadata like the page title, description, and author so the site is easier for browsers, search engines, and assistive technology to understand.
